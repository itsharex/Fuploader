# Fuploader - 多平台视频自动上传工具

Fuploader 是一款专为内容创作者和开发者设计的多平台视频自动上传工具，帮助用户通过简洁的配置快速将视频内容分发到多个主流社交媒体平台，支持多账号管理和高效批量操作。

---

## ✨ 核心特性

### 🎯 已支持平台

| 平台 | 状态 | 说明 |
|------|------|------|
| **B站 (Bilibili)** | ✅ 已上线 | 支持视频上传、封面设置 |
| **抖音 (Douyin)** | ✅ 已上线 | 支持视频上传、AI推荐封面、标签设置 |
| **快手 (Kuaishou)** | ✅ 已上线 | 支持视频上传 |
| **百家号 (Baijiahao)** | ✅ 已上线 | 支持视频上传 |
| **小红书 (Xiaohongshu)** | 🧪 测试阶段 | 支持视频上传，反爬机制持续优化中 |
| **视频号 (Tencent)** | 🧪 测试阶段 | 支持视频上传，页面渲染优化中 |
| **TikTok** | 🧪 测试阶段 | 支持视频上传，国际化适配中 |

### 🚀 主要功能

- **📹 视频批量上传** - 支持同时向多个平台上传视频，大幅提升发布效率
- **👤 多账号管理** - 支持为每个平台配置多个账号，灵活切换
- **⏰ 定时发布** - 支持设置视频定时发布（部分平台功能完善中）
- **🖼️ 封面设置** - 支持自定义视频封面（部分平台支持AI推荐封面）
- **🏷️ 标签管理** - 支持为视频添加标题、描述和标签
- **📊 任务调度** - 智能任务队列，同平台任务串行执行，避免冲突
- **🔒 Cookie管理** - 安全的账号登录状态管理
- **📱 独立桌面应用** - 基于 Wails 框架的跨平台桌面应用，开箱即用

### 💪 产品优势

| 优势 | 说明 |
|------|------|
| 🔒 **纯本地运行** | 所有操作均在本地完成，视频文件、Cookie 数据不会上传到任何第三方服务器 |
| 🛡️ **数据安全** | 账号 Cookie 本地加密存储，支持自动登录，无需担心账号信息泄露 |
| 🌐 **无需代理** | 直接连接各平台官方接口，无需中间服务器转发，上传速度更快更稳定 |
| 🎨 **开箱即用** | 独立桌面应用，无需配置复杂环境，下载即可使用 |
| 🔧 **开发者友好** | 提供完整 API 接口，支持二次开发和自动化集成 |
| 📊 **透明可控** | 开源代码，所有上传逻辑可见，无隐藏行为 |

## 📋 系统要求

- **操作系统**: Windows 10/11、macOS、Linux
- **浏览器**: Chrome 或 Chromium（用于自动化上传）
- **存储空间**: 建议预留 500MB 以上空间

## 🛠️ 快速开始

### 方式一：使用独立应用（推荐）

1. 下载对应系统的可执行文件
2. 双击运行 `fuploader.exe`（Windows）或 `fuploader`（macOS/Linux）
3. 在界面中添加账号并配置平台 Cookie(只需要登录cookie自动保存)
4. 创建上传任务，选择视频文件和目标平台
5. 点击发布，等待任务完成

### 方式二：开发者模式

```bash
# 克隆项目
git clone https://github.com/fishimei/Fuploader.git
cd fuploader

# 安装 Go 依赖
go mod download

# 安装前端依赖
cd frontend
npm install

# 开发模式运行
cd ..
wails dev

# 构建生产版本
wails build
```

- 如遇上传失败，建议检查平台页面是否有更新

## 🔧 配置说明

### Cookie 获取

**直接在Fuploader中添加账号,会去对应平台的登录页面,登录后会自动保存cookie**

### 视频文件要求

- **格式**: MP4（推荐）、MOV、AVI
- **大小**: 根据各平台限制（通常最大 2-4GB）
- **分辨率**: 建议 1080p 或以上
- **封面**: JPG/PNG 格式，建议与视频同名
